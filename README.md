## âš¡ Project Abstract

This repository contains the complete hardware design, schematic capture, and simulation profiles for a **Pulse Width Modulation (PWM) Signal Generator**. The project is engineered using the **Cadence OrCAD** suite and verified through **PSpice** simulation.

The circuit is designed to provide a stable, variable duty-cycle output suitable for power electronics applications, motor control, and signal modulation tasks.

## âš™ï¸ Technical Specifications

* **Supply Voltage ($V_{CC}$):** 5V - 15V DC
* **Output Signal:** Rectangular Pulse Wave
* **Duty Cycle Range:** Variable (approx. 5% - 95%)
* **Simulation Domain:** Time Domain (Transient Analysis)
* **Design Tool:** OrCAD Capture CIS / PSpice A/D
